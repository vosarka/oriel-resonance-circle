### Shadow Loudness Index (SLI)

```
SLI(r) = PCS(r) × StateAmp × FacetAmp(r)
```

Where:
- **FacetAmp(r)** = average of FL(facet) across occurrences of r in Prime Set

### SLI Thresholds (for selection)
