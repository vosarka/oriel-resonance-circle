# PDF Schema: VRC Static Signature Codex (15-Page Flagship Report)

This schema defines the page-by-page layout, typographic grid, visual systems, and validation indicators for compiling the **ORIEL Static Signature Codex** report.

---

## 1. DESIGN & TYPOGRAPHY GRID

- **Trim Size**: US Letter ($8.5 \times 11$ inches) or A4, portrait.
- **Margins**: $0.75$ inches (54pt) on all sides.
- **Grid**: 12-column modular grid with $0.25$ inches (18pt) gutters.
- **Color Palette**:
  - **Primary (Field)**: Black `#000000` or Deep Obsidian `#0D0E10` (used for background canvas).
  - **Secondary (Accents)**: Vossari Gold `#D4AF37` / HSL `(45, 60%, 53%)` (used for coherence state, titles, active channels).
  - **Active Signal**: Vossari Cyan `#00F0FF` (used for conscious activations, defined centers).
  - **Body / Neutral**: Ivory `#F5F2EB` (used for body copy, tables, grid borders).
- **Typography Hierarchy**:
  - **Titles (H1)**: Outfit or Inter, 24pt, Gold, Bold.
  - **Subheadings (H2)**: Outfit or Inter, 14pt, Cyan, Semi-bold.
  - **Body Text**: Outfit or Inter, 10pt/14pt leading, Ivory, Regular.
  - **Status/Audit Code (Monospace)**: Fira Code or JetBrains Mono, 8pt, Ivory/Muted Gray.
