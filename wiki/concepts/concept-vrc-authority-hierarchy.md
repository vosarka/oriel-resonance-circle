# VRC Authority Hierarchy (Decision Compass)

Authority in the VRC system is the consistent inner compass for decision-making. It is determined by scanning defined centers in a strict priority order.

## Priority Order (from CANON MASTER)

1. **Solar Plexus Center** → Emotional Resonance
2. **Sacral Center** → Gut Response
3. **Spleen Center** → Instinctive Pulse
4. **Heart Center** → Will / Desire
5. **G Center** → Self-Direction
6. **None / Outer** (Reflectors only) → Lunar Cycle
7. **Environment** → For certain Mental Projectors (Head/Ajna defined with no motors defined below the Throat)

The highest-priority defined center in this list determines the person's Authority.

## Key Points

- This is a deterministic scan, not a holistic or intuitive process.
- Reflectors have a special "Lunar Cycle" authority because they have no defined centers.
- The hierarchy protects against mental decision-making when stronger somatic or emotional authorities are present.
